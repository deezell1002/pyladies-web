โปรดอ่าน http://www.pyladies.com/CodeOfConduct การสมัครมาร่วมกิจกรรมนี้แปลว่าคุณยอมปฏิบัติตามเงื่อนไขของ The PyLadies Code of Conduct เพราะเราตั้งใจที่จะสร้างสภาวะที่ปลอดภัย เปิดกว้าง และปราศจากการคุกคามต่างๆ สำหรับผู้ร่วมกิจกรรมทุกคน โดยไม่คำนึงถึง เพศสภาพ ความไร้ความสามารถ รูปลักษณ์ภายนอก สถานะทางสังคม ขนาดร่างกาย ชาติพันธุ์ เชื้อชาติ ประสบการณ์ อายุ และศาสนา โดยเราจะไม่ยอมให้การคุกคามเกิดขึ้นไม่ว่าทางใดก็ตาม นอกจากนั้นแล้วเรายังให้ความสำคัญกับความปลอดภัยของกลุ่มคนชายขอบมากกว่าสิทธิพิเศษกับกลุ่มคนใดกลุ่มหนึ่งอีกด้วย
